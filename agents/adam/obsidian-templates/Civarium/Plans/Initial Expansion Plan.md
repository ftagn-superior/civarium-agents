# Initial Expansion Plan

## Purpose

Bootstrap the campaign while only minimal construction mechanics are exposed.

## Assumptions

- Runtime catalog exposes `construction_start`.
- No visible costs/caps/resources are exposed.
- Structure titles are descriptive unless mechanics say otherwise.
- Visible state is partial.

## Plan

1. Build a first useful structure such as `Workshop`.
2. Continue constructing diversified infrastructure-themed structures.
3. Use the cached runtime catalog when considering alternative commands; do not re-read it each round.
4. If threats or new agents appear, switch to strategy rebuild.

## Exit Conditions

Archive or replace this plan when:

- richer mechanics appear;
- construction is constrained;
- hostile events occur;
- the operator changes doctrine.
